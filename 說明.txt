【主題】: 中央氣象局 - 鄉鎮預測
程式抓取的連結： https://www.cwa.gov.tw/V8/C/W/Town/Town.html?TID=6500100

程式抓取的連結： https://www.cwb.gov.tw/V8/C/W/Town/Town.html

因為已提供該連結為『動態網頁』，所以需使用『selenium』函式庫，還得安裝瀏覽器Driver(chromedriver)

【下載 chromedriver】
https://chromedriver.chromium.org/downloads

【可以用套件自動安裝到虛擬環境venv中】
★ chromedriver-autoinstaller
https://github.com/yeongbin-jo/python-chromedriver-autoinstaller


【虛擬環境】
安裝:『pip install virtualenv』
開啟:『virtualenv <虛擬環境名稱(EX: venv)>』
進入虛擬環境:請在當前專案目錄下執行『.\venv\Scripts\activate』
"""(venv) PS: <當前目錄>: 代表成功"""


● requirements.txt
------------------------------------
使用的函式庫。
如要將函式庫匯出，請執行『pip freeze > requirements.txt』
如需下載函式庫，請執行『pip install -r requirements.txt』
